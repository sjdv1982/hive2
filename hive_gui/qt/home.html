The Hives tree-view panel can be used to add Hive nodes to the current editor space

Use the File menu to open, save or create new hive maps

Projects

Hive files can be collated under a project. The hives tree-view panel shows hive-maps from the current project.

Use the Open Project File menu option to change the current project. Existing project files will be closed.

Folding

Pull-in pins can be folded in to use an internal variable hive. Folded pins can be unfolded to expose these variables.

Cut Connections

Holding CTRL + LEFT MOUSE and dragging the cursor will draw a red slice line. Connections intersecting with this slice will be deleted.

Reorder Connections

Hovering over an output socket will show the indices of each connection. Sockets are connected from lowest to highest index, which is important for push connections. Select a connection and press [+] or [-] to change the connection index.
