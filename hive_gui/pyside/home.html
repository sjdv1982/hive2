The left tree-view panel (Hives ) can be used to add Hive nodes to the current node editor

Use the File menu to open, save or create new hive maps

Folding

Pull-in pins can be folded in to use an internal variable hive. Folded pins can be unfolded to expose these variables.

Cut Connections

Holding CTRL + LEFT MOUSE and dragging the cursor will draw a red slice line. Connections intersecting with this slice will be deleted.
